Contributors: hakubi-git
Requires at least: 6.0
Tested up to: 6.5
Stable tag: 1.1.3

== Description ==
Revify開発　SWELL用ブロックパターン集です。
このプラグインを有効にすると、ブロックエディタの「パターン」からRevifyオリジナルのデザインを呼び出せます。
